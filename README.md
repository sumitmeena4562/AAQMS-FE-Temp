# AI Safety Management System (AAQMS) 🛡️

**AI Safety Management System (AAQMS)** is a high-quality frontend application built for intelligent workplace monitoring and safety management. It provides a premium user experience with advanced data visualization and real-time alerts.

---

## ✨ Key Features
- **Elite User Interface**: Clean, professional design with a "Diamond Standard" aesthetic.
- **Role-Based Access**: Specialized flows for Admin, Coordinator, and Field Officers.
- **Smart Forms**: Advanced validation using Zod and React Hook Form.
- **Live Animations**: Powered by Framer Motion for a smooth and responsive feel.
- **Secure Authentication**: Protected routes and JWT simulation.
- **Safety Monitoring**: Dedicated modules for risk analysis and alerts.

---

## 🛠️ Technical Stack
- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Forms**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Icons**: [Lucide React](https://lucide.dev/) (and custom SVGs)

## 🚀 Getting Started

### Prerequisites
Make sure you have **Node.js** installed on your system.

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd AAQMS-FE
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📂 Project Structure
- `src/assets`: Global styles, icons, and theme variables.
- `src/components/ui`: Atomic UI components (Buttons, Inputs, Cards).
- `src/pages`: Feature-based page components.
- `src/layouts`: Master structures for Auth and Dashboard pages.
- `src/schema`: Data validation schemes for forms.
- `src/routes`: Application navigation logic.

---

## 📜 Available Scripts
- `npm run dev`: Start development server.
- `npm run build`: Build the app for production.
- `npm run lint`: Run ESLint to find and fix code issues.
- `npm run preview`: Preview the production build locally.

## 🎨 UI Guidelines
We follow a premium design language.
- Use **CSS Variables** defined in `index.css` for consistent colors and radius.
- Maintain **responsive layouts** using Tailwind and Flex/Grid.
- Ensure all interactive elements have subtle **motion effects**.
